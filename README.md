# AI KAPT

Локальный проект обучения ИИ для `дурака` с режимами `подкидной` и `переводной`.

## Запуск

1. Установить зависимости:
   - `npm install`
2. Запустить:
   - `start.bat`

## Стек

- Backend: Node.js + TypeScript + Express + Socket.IO
- Frontend: React + Vite + TypeScript
- Обучение: TensorFlow.js
